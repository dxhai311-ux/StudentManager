# Student Manager CLI

Simple Python CLI application for managing students.

## Features

* Show all students
* Add new student
* Calculate average score
* Find top student
* Input validation with try-except

## Technologies

* Python
* OOP (Object-Oriented Programming)
* Git & GitHub

## Project Structure

```text
student-manager-cli/
│
├── main.py
├── README.md
```

## Run Project

```bash
python code.py
```

## Example Menu

```text
===== Student Manager =====
1. Show students
2. Add student
3. Average score calculation
4. Top student
5. Exit
```
